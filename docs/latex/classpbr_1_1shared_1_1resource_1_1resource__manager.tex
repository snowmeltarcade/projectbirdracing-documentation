\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class pbr\+::shared\+::resource\+::resource\+\_\+manager$<$ T $>$}

Manages resources that need allocating and deallocating. When a resource is requested, if it does not exist, it is loaded. When it is returned to the caller, a reference count to it is increased. When that resource is no longer needed, it is returned to this manager. A reference count to it is then decreased. When the reference count is equal to zero, it is deallocated. Resources are loaded using the data manager.
