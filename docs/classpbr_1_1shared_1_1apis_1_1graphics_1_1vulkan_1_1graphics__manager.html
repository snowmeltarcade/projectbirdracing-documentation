◆ submit_frame_for_render()

void pbr::shared::apis::graphics::vulkan::graphics_manager::submit_frame_for_render ( )
overridevirtualnoexcept

Submits a frame for rendering. In Vulkan, this simply submits a request for the frame to be rendered, the frame itself will be rendered when the driver is ready. If no color frames are free to submit a render request to, the thread will block waiting for an image to become free. All previously submitted renderable entities will be cleared after this frame is submitted
